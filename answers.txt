Ответ: Топология «звезда».

Ответ: Коммутаторы (switch), маршрутизаторы (router), сетевые адаптеры.

4. Какая категория кабеля обычно используется для подключения компьютеров к локальной сети?
Ответ: Категория Cat5e или Cat6.

5. Какой стандарт обычно используется для установки сетевых устройств в офисной среде?
Ответ: Стандарт Ethernet (IEEE 802.3).

6. Какая скорость передачи данных обычно используется в локальной сети?
Ответ: 100 Мбит/с, 1 Гбит/с, иногда 10 Гбит/с.

7. Какой стандарт обычно используется для безопасности локальной сети?
Ответ: IEEE 802.1X.

8. Какая технология обычно используется для разделения сети на виртуальные сегменты?
Ответ: VLAN.

9. Какие устройства обычно используются для управления трафиком в локальной сети?
Ответ: Маршрутизаторы и управляемые коммутаторы.

10. Какая структура кабельной системы обычно используется для монтажа локальной сети?
Ответ: Структурированная кабельная система (СКС).

11. Какой тип кабельных коннекторов обычно используется для подключения компьютеров к сети?
Ответ: RJ-45.

12. Какая система маркировки кабелей обычно используется для облегчения их управления и обслуживания?
Ответ: Бирки и цветовая маркировка кабелей по стандартам СКС.

13. Какая категория защиты от пыли и влаги обычно используется для установки сетевых устройств в офисной среде?
Ответ: IP20.

14. Какой тип маршрутизатора обычно используется для выделения отдельной сети в локальной вычислительной сети?
Ответ: Маршрутизатор с поддержкой NAT и VLAN.

15. Какая технология обычно используется для обеспечения отказоустойчивости в локальной сети?
Ответ: Резервирование каналов и оборудования (STP, LACP).

16. Какая технология передачи данных обычно используется для монтажа локальной сети?

17. Какой стандарт обычно используется для настройки устройств в локальной сети?
Ответ: TCP/IP.
